    - Generic connection errors should return 503.
  </behavior>
  <action>
    - Wrap all `httpx` calls in `try/except` blocks in the following files:
      - `backend/services/ncbi.py`
      - `backend/services/ensembl.py`
      - `backend/services/vep.py`
      - `backend/services/regulomedb.py`
    - Catch `httpx.HTTPStatusError`, `httpx.TimeoutException`, and `httpx.ConnectError`.
    - Re-raise as `fastapi.HTTPException` with appropriate status codes (503 for Service Unavailable, 504 for Gateway Timeout).
    - Create `backend/tests/test_api_resilience.py` to mock external API failures and verify the backend response.
  </action>

  </verify>

<objective>
Optimize the backend variant annotation pipeline to handle thousands of variants efficiently through increased batching and optimized discovery limits. This plan lays the foundation for "high-density" variant discovery by ensuring the discovery phase is robust and performant.

Key Improvements:
- VEP batch size: 200 -> 1000 (per D-03)
- RegulomeDB: Single-rsid -> Batch-rsid lookup
- Discovery Limit: 500 -> 2000 variants per gene
- DB Indexing: Add indices for impact and consequence
</objective>

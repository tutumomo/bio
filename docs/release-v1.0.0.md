# v1.0.0 — 首個正式版本上線

Helix Bio v1.0.0 是本專案的首個正式版本，帶來完整可用的基因／變異查詢流程，從基因搜尋、變異篩選，到結果瀏覽與歷史追蹤，已具備穩定的實際使用基礎。

## 亮點功能

- 基因與蛋白搜尋
  - 支援以基因名稱進行查詢，快速取得基因基本資訊與相關結果。

- 高效能變異篩選
  - 可依多種條件篩選變異資料，包含影響程度、功能後果與分數區間等常見分析維度。
  - 查詢與分頁體驗已針對實際使用情境優化。

- 更完整的結果頁體驗
  - 搜尋結果與篩選條件可同步到網址，方便分享、重整後保留狀態，以及回到先前分析情境。
  - Gene Detail 與 Results 頁面流程更一致，減少切換成本。

- 持久化登入與使用者體驗提升
  - 支援登入狀態持久化，減少重複登入。
  - 使用者相關流程已整合到資料庫模型，帳號體驗更穩定。

- 搜尋歷史
  - 支援查詢歷史紀錄、分頁瀏覽，以及依條件重播先前搜尋。
  - 更適合需要反覆比較或回顧分析結果的使用情境。

- 進階資料整合
  - 加入表現量視覺化與額外資料來源整合，讓單一查詢能提供更完整的參考資訊。

## 本版本包含

- 首波核心查詢體驗完成
- 身分驗證與使用者資料持久化
- 搜尋歷史與回放能力
- 生產環境設定與部署相關調整
- 整體穩定性與產品細節打磨

## 適合誰使用

這個版本適合希望快速查詢基因與變異資訊、進行初步篩選分析、並保留查詢脈絡的使用者。若你需要一個可實際操作、可持續回顧結果的 Bioinformatics Web App，`v1.0.0` 已可作為正式起點。

## 備註

若你是自行部署的使用者，請確認後端資料庫、JWT 與 OAuth 相關環境變數已正確設定，前端也需配置對應的 API URL。
